# Sentiment-analysis-for-IMDB-movies
Sentiment analysis on IMDB movie reviews (positive/negative). Tried three approaches: hand-crafted features, Bag-of-Words with TF-IDF, and BERT embeddings. Compared classifiers (logistic regression, SVM, SGD) and evaluated with F1, balanced accuracy, and ROC-AUC. BERT performed best at ~88%.
